# Fonts

請把你要給 Blink Shell 使用的字型檔放在這裡。

目前這份 repo 內的 CSS 預設期待以下檔名：

- `SarasaMonoTC-Regular.ttf`
- `SarasaMonoTC-Bold.ttf`

## 字型下載來源

從 [Sarasa Gothic 官方 Releases](https://github.com/be5invis/Sarasa-Gothic/releases) 下載最新版本，  
取出 `sarasa-mono-tc-regular.ttf` 及 `sarasa-mono-tc-bold.ttf`，並依上方檔名重新命名後放入此目錄。

## 授權說明

Sarasa Gothic 字型採 [SIL Open Font License 1.1 (OFL)](https://scripts.sil.org/OFL) 授權。  
若要將字型檔公開放在 GitHub，建議一併保留上游授權檔案（`LICENSE-SIL-OFL.txt`）。

## 擴充字重

如果你之後想加 `Medium`，再同步更新 `blink/sarasa-mono-tc-blink.css` 即可。
